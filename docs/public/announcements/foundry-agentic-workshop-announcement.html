Announcing · Hands-on Lab

Microsoft Foundry
Agentic Workshop

A partner-ready, facilitator-led hands-on lab that takes engineers from zero to production on the latest Microsoft Foundry Agent Service capabilities — including features announced at Build 2026.

The lab deploys shared Azure infrastructure to the partner's tenant and guides lab attendees through building agents, wiring tools, and running evaluations in a shared Foundry environment.

View Documentation GitHub Repository
3–4 Hours End-to-End 12 Modules Build 2026 Features Partner Ready Python + Azure Developer CLI
Why this lab, why now?
Microsoft Foundry Agent Service shipped a significant wave of capabilities at Build 2026 — Hosted Agents deployed from source code, Foundry Toolboxes, Foundry IQ knowledge bases, and MCP tool integration. This lab gives you a structured, repeatable way to take partners through every capability hands-on in a shared Azure environment, in a single session.

How the lab works

🏗️
Organizer
Deploy the shared environment
Run azd provision to stand up a shared Foundry account, AI Search, Azure Container Apps MCP server, and Application Insights — all in a sandbox subscription. One command, one environment, zero manual portal work.
→
🎤
Facilitator
Present & guide attendees
Walk the group through each module using the facilitator guide. Attendees follow along in their own Foundry project — the facilitator keeps the room in sync, calls on proctors, and paces the session.
→
🧑‍💻
Attendees
Build in their own project
Each attendee gets a dedicated Foundry project pre-configured with their identity. They build agents, wire tools, and run code — with no risk of stepping on each other's work.

Who should run this?

This lab is designed to be run with partner technical teams — SIs, ISVs, and enterprise engineering groups — who are evaluating or building on Microsoft Foundry. It assumes working familiarity with Python and Azure, but no prior Foundry experience is required.
